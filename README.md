# Envanter_stok_uygulamas-
PyQt5 ve SQLite3 ile geliştirilmiş; dinamik kategori renklendirmeli, milimetrik porsiyon bazlı stok düşümlü ve WhatsApp/E-posta entegreli profesyonel kafe envanter ve tedarik yönetim otomasyonu.
güncel.py dokümanıyla yapılması planlanan uygulamanın temeli atılmış oldu.Yol haritası oluşturabilme açısından uygulama iskeleti oluşturuldu.
